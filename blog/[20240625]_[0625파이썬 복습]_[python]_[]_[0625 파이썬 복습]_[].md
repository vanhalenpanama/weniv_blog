# 람다함수의 기본구조

#람다 함수는 다음과 같은 형식을 따릅니다.
lambda 매개변수들: 표현식
# 매개변수 : 함수에 전달되는 입력값, 여러개일수 있음 쉼표로 구분할수 있습니다.
# 표현식 : 삼항연산자와 비슷!

#간단한 함수가 필요할 때
#다른 함수의 인자로써 간단한 함수를 전달할때
#가독성을 해치지 않는 범위내에서 코드의 간결성을 추구할때

# 1-1. 한개의 매개변수를 가지는 람다함수

double = lambda x: x * 2 # 파라미터 -> 계산식
print(double(3))

# 1-2. 1-1을 풀어서 쓰면?
def double_1(x):
    return x * 2
print(double_1(3))


# 2-1. 두개의 매개변수(파라미터)를 받는 람다함수
add = lambda x, y: x + y
print(add(3, 4))


# 2-2.2-1을 풀어서 쓰면?

def add_1(x, y):
    return x + y
print(add_1(3, 4))

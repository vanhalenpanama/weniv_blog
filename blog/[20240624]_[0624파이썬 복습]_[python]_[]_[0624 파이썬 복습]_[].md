파라미터 (Parameter)
파라미터는 함수를 정의할 때 사용되는 변수입니다. 파라미터는 함수가 호출될 때 들어온 값을 변수에 할당하는 역할을 합니다

아규먼트 (Argument)
아규먼트는 함수를 호출할 때 전달하는 실제 값입니다. 이 값들은 함수 호출 시 파라미터에 할당됩니다.






이 방법은 lst 파라미터의 기본값을 None으로 설정하고, 함수 본문 내에서 None 값을 새로운 리스트로 바꾸어 주어, 함수 호출 사이에 리스트가 공유되지 않도록 합니다. 이렇게 하면, 각 함수 호출이 독립적인 리스트를 가지게 되므로, 위에서 설명한 예상치 못한 부작용을 피할 수 있습니다.

# 권장 방법
def append_to_list(value, lst=None):
    if lst is None:
        lst = []
    lst.append(value)
    return lst

지역 변수와 전역 변수


1. 지역 변수
지역 변수는 함수 내부에서만 정의되고 사용되는 변수입니다. 이를 통해 다른 함수나 프로그램의 전체 흐름에 영향을 주지 않고, 함수 내부에서만 데이터를 관리할 수 있습니다.

# 지역 변수 사용 예시
def f():
    a = 1
    print(a)  # 출력: 1


2. 전역 변수
전역 변수는 프로그램 전체, 어떤 함수에서도 접근 가능한 변수입니다. 함수 내부에서 전역 변수를 사용하려면 전역 변수와 동일한 이름으로 앞에 global로 선언해야됩니다.

빌트인 함수


all() 함수는 그 안에 있는 모든 요소가 True로 평가되면 True를 반환하고, 그렇지 않으면 False를 반환합니다.

any() 함수는 그 안에 있는 어떤 요소라도 True로 평가되면 True를 반환하고, 그렇지 않으면 False를 반환합니다.



chr() 함수는 정수를 인수로 받아 그에 해당하는 유니코드 문자를 반환합니다.
chr(65) # 출력: 'A'
chr(97) # 출력: 'a'



ord() 함수는 문자를 인수로 받아 그 문자의 유니코드(또는 ASCII 코드) 값을 반환합니다.
ord('a') # 출력: 97
ord('하') # 출력: 54616






map(함수, 리스트 혹은 튜플)
map객체 자체로는 쓸 일이 없을 것 같다.(단독으로 출력시 map object at 0x******* 이렇게 메모리 주소를 반환함)

리스트 또는 튜플로 감싸서 써야될듯

# 첫번째 방법
def 제곱(x):
    return x ** 2
list(map(제곱, [1, 2, 3, 4])) # 출력: [1, 4, 9, 16]

# 두번째 방법
list(map(lambda x : x ** 2, [1, 2, 3, 4]))





filter(조건 함수, 리스트 또는 튜플)

list(filter(lambda x : x > 50, range(100)))






zip()
 여러 개의 순회할 수 있는 객체의 요소를 묶어서 반환합니다. 

zip(순회할 수 있는 객체1, 순회할 수 있는 객체2...)

list(zip('ab', '12', [10, 20])) 
# 출력: [('a', '1', 10), ('b', '2', 20)]


print(dict(zip('ab', '12')))
# 출력 : {'a': '1', 'b': '2'}








format() 함수는 특정 값을 지정된 형식으로 변환하여 문자열로 반환하는 함수입니다. 만약 서식 지정자를 설정하지 않으면 str()한 것과 동일한 결과를 얻을 수 있습니다.


format(값, '서식 지정자(format_spec)')


format(10000, ',') # 10,000

print(format(123.4567, '.2f')) # 출력: 123.46
print(format('Hello', '^9')) # 출력: '  Hello  '








sum 함수는 순회 가능한 객체를 받아 모든 요소의 합을 계산하여 반환합니다. 

sum([1, 2, 3, 4, 5]) # 출력: 15
sum([1, 2, 3], 10) # 출력: 16




len(길이를 계산하려는 객체)




max 함수는 주어진 순회할 수 있는 객체의 최댓값을 반환합니다.

기본적인 형태는 아래와 같습니다.

max(순회할 수 있는 객체, 함수(key), 초깃값(default))


max([1, 2, 3, 4, 5]) # 출력 : 5
max([(1, 2), (3, 101), (1, 20), (3, 100)]) # 출력 : (3,101)
max(['hello', 'world', 'world1']) # 출력 : world1
